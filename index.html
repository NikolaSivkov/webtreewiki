@Templates.RenderTemplate()

Renders a template that doesn not depend on model

Parameters
Name Type Description
menu string name of the menu template to use
folder string name of the folder to look in for the specified template default is 'shared'
Example
                        <html>
                            <head>
                            </head>
                            <body>
                                @Templates.RenderTemplate("_menu")
                            </body>
                        </html>
                        

                        <html>
                            <head>
                            </head>
                            <body>
                                @Templates.RenderTemplate("footer","other")
                            </body>
                        </html>

@Templates.RenderPages()

Render a list of pages.

Parameters
Name Type Description
templateName string renders the "list.cshtml" template in the "/Page/" folder, when set looks for template with the set name in the "/Page/" folder
amount int Set how many pages to fetch. Default - 20
offset int Set the number of pages to skip. Default - 0
Example
                        <html>
                            <head>
                            </head>
                            <body>
                               @Templates.RenderPages("MyPagesTemplate")
                            </body>
                        </html>
                        

                        <html>
                            <head>
                            </head>
                            <body>
                              @Templates.RenderPages("MyPagesTemplate",20)
                            </body>
                        </html>
                        
                        <html>
                            <head>
                            </head>
                            <body>
                              @Templates.RenderPages("MyPagesTemplate",20,10)
                            </body>
                        </html>
